# aoc2023

This is a collection of my solutions for the [Advent of Code 2023](https://adventofcode.com/2023) challenge written in [Go](https://go.dev/)


## Running
``` shell
go run main.go <day>
```
### Example
This will run the solution for day 1
``` shell
go run main.go 1
```
